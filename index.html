Spatial Canvases

Assignment 3 — 2D (p5.js) and 3D (three.js) Spatial Canvases

Design Statement

This work explores the translation of a simple hand-drawn gesture — a single stroke, a filled circle, a scribbled outline — into procedural form. The 2D sketches treat the mark as raw material: a fixed vocabulary of primitives arranged into a composed scene, and a live, responsive trace that keeps changing as it is made. The 3D sketches carry that same idea into space, using primitive geometry, layered materials, light, and fog to give the marks volume, depth, and atmosphere. Underlying all four pieces is an interest in where the boundary sits between an authored gesture and a generative system that elaborates on it.

3D Spatial Canvases — three.js

01 — Procedural Geometry, Orbit Camera

A rotating field of primitive geometries (spheres, cones, tori, icosahedra) arranged along a spiral, viewable from any angle with an orbiting camera.

02 — Materials, Lighting & Fog

A small landscape composition exploring standard, physical (glass-like water), and emissive materials under directional + ambient light, with exponential fog for depth.

2D Spatial Canvases — p5.js

03 — Static Composition, Primitive Shapes

A fixed abstract landscape built entirely from circles, arcs, triangles, and lines, using a limited, considered color palette.

04 — Interactive / Animated Drawing

Draw with the mouse — each stroke spawns a trail of drifting, fading particles. Click "clear" to start over.

Built with p5.js and three.js — code available in this repository.
